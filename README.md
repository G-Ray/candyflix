Installation
-------

1. Install Node.js
2. Install peerflix
3. Serve the frontend folder with your favorite webserver
4. Set a proxypass to redirect from /socket.io/ to http://localhost:3000
5. The fun part :

```
cd app
npm install
node candyflix.js
```
